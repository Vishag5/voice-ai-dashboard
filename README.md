# 🎙️ VoiceAI Portfolio Dashboard

A high-fidelity prototype of an AI Data Operations Platform designed for Product Managers. Built to showcase expertise in speech workflows, model evaluation metrics (WER, CER, DER), and Human-in-the-Loop (HITL) systems.

## 🚀 Live Demo Instructions

To view this dashboard publicly:
1.  **Fork/Clone** this repository.
2.  Go to **Settings** > **Pages**.
3.  Select **Deploy from a branch** and choose `main`.
4.  Your dashboard will be live at `https://<your-username>.github.io/<repo-name>/`

## 🛠️ Features

### 1. Voice Annotation Studio
*   **Synced Waveform**: Visual signal representation synced with audio playback.
*   **Metadata Tagging**: Categorization for background noise, speaker accent, and confidence.
*   **Transcription Logic**: High-utility UI for speed and accuracy.

### 2. Video Perception Studio
*   **Object Detection**: Simulated bounding boxes with confidence scores (IoU).
*   **Class Management**: Tracking instances of detected objects in real-time.

### 3. QA & Consensus Resolution
*   **Conflict Detection**: Flags disagreements between multiple annotators (e.g., medical terminology).
*   **Reviewer Pipeline**: Senior-level resolution flow for ground-truth data delivery.

### 4. PM Analytics
*   **Model Eval**: Tracking WER, CER, and DER across model versions (v1.8 vs v2.0).
*   **Throughput tracking**: Visualizing the flow from Collection → Annotation → QA → Delivery.

# Step-by-Step GitHub Deployment Guide

Follow these steps to make your **VoiceAI Dashboard** public and shareable with anyone in under 5 minutes.

### 1. Create a GitHub Account
If you don't have one, sign up at [github.com](https://github.com).

### 2. Create a New Repository
1.  Click the **+** icon in the top right and select **New repository**.
2.  Name it: `voice-ai-dashboard`.
3.  Set it to **Public**.
4.  Do **NOT** initialize with a README (we already have one).
5.  Click **Create repository**.

### 3. Upload Your Files
1.  On your computer, open the folder: `c:\Users\USER\Downloads\Companion App\Agent\voice-ai-dashboard`.
2.  On the GitHub "Quick Setup" page, click the link that says "**uploading an existing file**".
3.  Drag and drop the following files into the GitHub window:
    *   `index.html`
    *   `style.css`
    *   `app.js`
    *   `README.md`
4.  Click **Commit changes** (wait for it to finish).

### 4. Enable GitHub Pages
1.  In your GitHub repository, click on the **Settings** tab.
2.  On the left sidebar, click **Pages**.
3.  Under **Build and deployment** > **Branch**, click the dropdown and select `main`.
4.  Click **Save**.

### 5. Get Your Public Link
Wait about 60 seconds. Refresh the **Pages** settings page. You will see a banner that says:
> **"Your site is live at https://<your-username>.github.io/voice-ai-dashboard/"**
